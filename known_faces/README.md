# Known Faces Directory

Add photos of people you want to recognize here.

## Instructions:
1. Add clear, front-facing photos of people
2. Name files as: `PersonName.jpg` or `PersonName.png`
3. Examples:
   - `John.jpg`
   - `Sarah_Smith.png`
   - `Alice.jpeg`

## Tips:
- Use clear, well-lit photos
- One face per image
- Front-facing photos work best
- Higher quality images = better recognition
